About InfluxDB

InfluxDB is a time series, events, and metrics database. It’s written in Go and has no external dependencies. That means once you install it there’s nothing else to manage (like Redis, HBase, or whatever). It’s designed to be distributed and scale horizontally, but be useful even if you’re only running it on a single box. There are three of us (Paul Dix, Todd Persen, and John Shahid working full time on the core of the database. We’re a YC (W13) backed company.
